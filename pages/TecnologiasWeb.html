Tecnologías Web: HTML, CSS y Bootstrap

Descubre cómo estas tecnologías forman la base del desarrollo frontend moderno.

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado fundamental que define la estructura y el contenido de una página web. Piensa en HTML como los "huesos" de una página: sin él, no habría dónde colocar el texto, las imágenes o los videos. Sus etiquetas, como <h1> para títulos y <p> para párrafos, organizan el contenido de forma semántica, haciendo que la página sea legible tanto para los navegadores como para los motores de búsqueda.

Ejemplo básico de HTML:

<!DOCTYPE html>
<html lang="es">
<head>
<title>Mi primera página</title>
</head>
<body>
<h1>Título principal</h1>
<p>Este es un párrafo de ejemplo.</p>
<ul>
<li>Elemento 1</li>
<li>Elemento 2</li>
</ul>
</body>
</html>

CSS

CSS (Cascading Style Sheets) es el "estilista" de la web. Mientras HTML se encarga de la estructura, CSS se ocupa de la apariencia visual: colores, tipografía, márgenes, animaciones, y cómo se ve la página en diferentes dispositivos. Gracias a CSS, podemos transformar una página HTML básica en una experiencia visual atractiva y profesional.

Ejemplo de cómo un archivo CSS estiliza la página:

body {
font-family: 'Montserrat', sans-serif;
background-color: #f0f4f8;
}
h1 {
color: #2c3e50;
text-align: center;
}
p {font-size: 1.1em;
 line-height: 1.6;
}

Bootstrap

Bootstrap es un framework CSS que acelera enormemente el desarrollo web. Proporciona una colección de plantillas y componentes predefinidos (como menús de navegación, botones, carruseles, etc.) que puedes usar con solo añadir clases CSS a tus elementos HTML. Su característica más poderosa es el **diseño responsivo** de forma predeterminada, lo que significa que los sitios construidos con Bootstrap se ven bien en cualquier dispositivo, ya sea un móvil, una tablet o un monitor de escritorio.

Ejemplo de uso de Bootstrap:

<button class="btn btn-primary">Botón Bootstrap</button>
<div class="row">
<div class="col-md-6">Columna 1</div>
<div class="col-md-6">Columna 2</div>
</div>

Cómo utilizar HTML, CSS y Bootstrap juntos

El flujo de trabajo para crear una página web moderna típicamente combina estas tres tecnologías:

  1. Estructura con HTML: Primero, se crea el esqueleto de la página con etiquetas HTML. Se definen el encabezado, el cuerpo, secciones, párrafos, imágenes, etc.
  2. Estilo con CSS: Luego, se utiliza CSS para dar vida a esa estructura. Se definen los colores, la tipografía y el diseño, creando un archivo .css aparte para mantener el código limpio y organizado.
  3. Componentes y Diseño Responsivo con Bootstrap: Finalmente, para acelerar el proceso y asegurar que el diseño sea responsivo, se añaden las clases de Bootstrap. Por ejemplo, en lugar de escribir todo el CSS para un botón, simplemente se añade la clase .btn y .btn-primary a un elemento <button>. Esto crea un botón con un estilo predefinido y listo para usar en cualquier dispositivo.

La combinación de estos tres elementos permite a los desarrolladores crear páginas web atractivas, funcionales y adaptables de manera eficiente.
